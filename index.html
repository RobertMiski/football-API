Hello, my name is Visan Alexandru and I decided to make this website because all sites that have information about football matches have ADS,so this is an ad-free standing and match information site

As you can see in the right corner there is a button named "STANDINGS" if you click on the button you can see there are 6 leagues that you can choose from.

As you can see in the left corner there is a menu if you click on the menu you can see there are 6 leagues that you can choose from and all matches from that specified league.

Every league it's different,it has competitions with: promotions to UCL-UEL-ULC and relegations, and it also has Championship with the playoff type

And as you can see on the left side, you can see "MATCHES" if you click on the links it will bring you to another page with all the games recently played in that league
